## 게임 설명
10x10 맵에서 몬스터를 피해 금(G)에 도달하면 승리하는 게임

## 조작 방법
- w: 위
- s: 아래
- a: 왼쪽
- d: 오른쪽

## 등장 요소
- #: 플레이어
- M: 몬스터 (플레이어 추적)
- G: 금 (목표 지점)

## 개발 환경
- Visual Studio 2026
- C언어
