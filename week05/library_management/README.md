## 📖 **프로젝트 개요**

**C 언어 콘솔 기반 도서관 관리 시스템**

도서 CRUD(Create, Read, Update, Delete) 기능 구현
최대 50권 고정 배열 사용
안전한 문자열 입력 처리

---

## ✨ **주요 기능**

| #️⃣    | 기능          | 입력             | 출력                                |
| ---- | ------------- | ---------------- | ----------------------------------- |
| 1    | **ID 검색**   | `0`              | `책이름 = 해리포터\n저자 = J.K롤링` |
| 2    | **저자 검색** | `킹`             | `ID: 3 | 반지의 제왕 | 톨킨`        |
| 3    | **제목 검색** | `포터`           | `ID: 0 | 해리포터 | J.K롤링`        |
| 4    | **책 추가**   | `해리포터`, `JK` | `책이 추가 되었습니다.`             |
| 5    | **전체 목록** | `-`              | `총 도서 수: 3\nID: 0 | ...`        |
| 6    | **종료**      | `6`              | 프로그램 종료                       |

## 🛡️ **안전성 구현**

✅ 버퍼 오버플로우 방지
\- name → "%19s"
\- author → "%9s"

✅ 배열 범위 검사
\- ID < 0 || >= count_book
\- 최대 50권 제한

✅ 정상 종료

## 🧠 **배운 점** ⭐

| **주제**          | **처음 생각** | **문제 발견**         | **해결 & 배운 점**                                           |
| ----------------- | ------------- | --------------------- | ------------------------------------------------------------ |
| **문자열 입력**   | `scanf("%s")` | 긴 입력 → 메모리 파괴 | **`배열[N] → "%(N-1)s"`**로 `\0` 확보                        |
| **배열 범위**     | 번호만 검사   | 음수/초과 접근        | **`< 0 || >= 크기`** 항상 검사                               |
| **`&` 사용 차이** | 모든 변수에 & | 배열은 안 붙임        | **`"배열 이름 = 첫 번째 원소 주소"`**<br>`char name[20]` → `scanf("%s", name)`<br>`int num` → `scanf("%d", &num)` |
| **메모리 관리**   | 구조체 배열   | 스택 메모리           | **고정 배열 = 효율적** (1.7KB)                               |
| **프로그램 종료** | `exit(1)`     | 비정상 종료           | **`return 0;`** = 성공 의미                                  |
