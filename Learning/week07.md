## 이번 주 주요 내용

- 핵심 주제:  파일 입출력
- 목표 학습 포인트:
  - fopen, fclose을 이용한 파일 열고 닫기
  - fgetc, fputc을 이용한 파일 읽고 쓰기
  - fseek, ftell을 이용한 파일 포인터 변경

---

## 강의 핵심 개념

### 1. 스트림

- 정의: 데이터를 버퍼 단위로 나누어서 전달하는 방식
- 특징 / 특성: c언어에서는 대부분의 입출력이 스트림 방식으로 저장되어 있음
- 예시 (필요 시): 파일 입출력, 표준 입출력 

### 2. 텍스트 파일

- 정의: 인간이 볼 수 있게 저장된 파일
- 특징 / 특성: 라인이 나뉘어짐, 아스키 코드를 이용함
- 예시 (필요 시): c 프로그램 소스 파일, 메모장 파일 등등

### 3. 이진 파일

- 정의: 컴퓨터가 읽을 수 있게 저장된 프로그램
- 특징 / 특성: 이진수로만 저장됨, 특정 프로그램에 의해서만 판독 가능
- 예시 (필요 시): c프로그램 실행 파일, 사운드 파일, 이미지 파일

---

## 실습 / 과제 요약

- 실습 주제: 파일 입출력 함수를 이용한 프로그램 개발
- 사용한 도구 / 언어: 비주얼 스튜디오 2026 / C언어
- 핵심 코드/수식 (필요 시): fgetc 함수는 int 값을 반환한다 (변수 타입 int 형으로 선언)

```c
#include <stdio.h>

int main() { //a.txt 내용을 b.txt에 복사하는 프로그램 a를 읽고 b의 쓴다
	FILE* fp1, *fp2;
	char text;
	fp1 = NULL;
	fp2 = NULL;
	fp1 = fopen("a.txt","r");
	fp2 = fopen("b.txt","w");
	if (fp1 == NULL || fp2 == NULL) {
		printf("파일을 찾을 수 없습니다.");
	}
	while ((text=fgetc(fp1)) != EOF) {
		fputc(text,fp2);
	}
	fclose(fp1);
	fclose(fp2);
	return 0;
}

	

```
